# 🏋️ Project MODEL FITNESS
**Análisis Predictivo de Cancelación de Membresías en Gimnasios**

## 📋 Descripción del Proyecto
**PROJECT MODEL FITNESS** es un análisis integral de datos desarrollado para una reconocida cadena de gimnasios, con el objetivo de **predecir la cancelación de membresías** y **mejorar la retención de clientes**.  

El proyecto incluye:  
- Exploración y limpieza de datos  
- Análisis de correlaciones y patrones de comportamiento  
- Construcción de modelos de **machine learning** para predicción de churn  
- Segmentación de usuarios mediante **clustering**  

---

## ✨ Características Principales
- 🔍 **Data Exploratory Analysis (EDA):** identificación de tendencias clave en el comportamiento de clientes  
- 🧹 **Limpieza de Datos:** tratamiento de valores nulos y outliers  
- 📈 **Modelos Predictivos:** regresión logística y random forest  
- 🤖 **Machine Learning:** precisión superior al **91% en predicción de cancelaciones**  
- 👥 **Segmentación de Clientes:** clustering para perfilar niveles de riesgo  
- 🎯 **Recomendaciones de Retención:** estrategias personalizadas por segmento  

---

## 🎯 Conclusiones
- **Predicción Efectiva:** El modelo random forest logró un desempeño destacado en la identificación de clientes en riesgo.  
- **Segmentación Estratégica:** El clustering permitió distinguir perfiles de usuarios con diferentes necesidades.  
- **Acción Personalizada:** Los hallazgos ayudan a crear programas de fidelización segmentados y más efectivos.  
- **Optimización de Negocio:** Se ofrece un marco sólido para la toma de decisiones orientadas a retención y satisfacción del cliente.
